# EcoTrack – Smart Waste Management Android Application

EcoTrack is an Android application developed using Kotlin and Firebase to simplify waste reporting and pickup management. The application allows users to report waste issues, request waste pickup, and track report status in real time.

## Features

- User Signup and Login using Firebase Authentication
- Waste Reporting System
- Pickup Request Module
- Real-time Firebase Realtime Database Integration
- RecyclerView for Dynamic Waste Report Display
- Report Status Tracking
- Logout Functionality
- Multi-Activity Navigation
- Responsive XML-based UI Design

## Tech Stack

- Kotlin
- XML
- Android Studio
- Firebase Authentication
- Firebase Realtime Database
- RecyclerView
